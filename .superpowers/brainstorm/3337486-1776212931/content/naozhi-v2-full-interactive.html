AWS WAF 知识汇编

核心发现

⚠ 高误拦截风险规则
AWSManagedRulesSQLiRuleSet 对 JSON body 误判。已切 Count 模式。SizeRestrictions_BODY 拦截大 payload，Zeelool 和 Giikin 受影响。
Sources: Dashboard "WAF Rule 优化", Feishu (4/8), CLI terraform-waf

Rate Limiting

100/5min → 200/5min 后误拦截率 3.2% → 0.4%。API heavy 用 IP + URI 组合限流。

Terraform 配置

waf.tfCopy
resource "aws_wafv2_web_acl" "main" {
  rule {
    name     = "AWSManagedRulesSQLiRuleSet"
    priority = 20
    override_action { // Count mode
      count {}
    }
  }
}
